# HEALIFY360 Backend

## Getting Started

-   Clone the repository

```
git clone git@github.com:herdeybayor/healify360-backend.git
or
git clone https://github.com/herdeybayor/healify360-backend.git
```

-   Change directory

```
cd healify360-backend
```

-   Install dependencies

```
npm install
or
yarn install
```

-   Start the server

```
npm start
or
yarn start
```
